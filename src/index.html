LC_Grind API Documentation

Company-wise Questions

Retrieve questions specific to a company.

Request Type: GET
Endpoint: /specific?companies={companyName}

Example:

/specific?companies=Google

Query Questions by Company

Query questions based on one or more companies.

Request Type: GET
Endpoint: /query?companies={companyName}

Example:

/query?companies=Google

Difficulty Levels

Retrieve questions based on difficulty.

  • Request Type: GET - Easy questions (Endpoint: /easy)
  • Request Type: GET - Medium questions (Endpoint: /medium)
  • Request Type: GET - Hard questions (Endpoint: /hard)

Examples:

/easy
/medium
/hard

Add User or Update Questions

Add a new user or update questions for an existing user.

Request Type: POST
Endpoint: /add

Example Request Body:

{ "user_id": "userId123", "question_ids": ["questionId1", "questionId2"] }

Delete Question

Delete a question for a specific user.

Request Type: POST
Endpoint: /delete-question

Example Request Body:

{ "user_id": "userId123", "question_id": "questionId1" }

Search

Search for user's questions.

Request Type: GET
Endpoint: /search?user_id={userId}

Example:

/search?user_id=userId123
