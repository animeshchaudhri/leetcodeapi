LC_Grind API Documentation

Company-wise Questions

Retrieve questions specific to a company.

/specific?companies={companyName}

Example:

/specific?companies=Google

Query Questions by Company

Query questions based on one or more companies.

/query?companies={companyName}

Example:

/query?companies=Google

Difficulty Levels

Retrieve questions based on difficulty.

  • /easy - Easy questions
  • /medium - Medium questions
  • /hard - Hard questions

Examples:

/easy
/medium
/hard
